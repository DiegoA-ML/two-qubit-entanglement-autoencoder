# two-qubit-entanglement-autoencoder
Deep learning pipeline for predicting two-qubit concurrence from Pauli expectation values using a two-stage autoencoder–regressor architecture. Implements dataset generation with toqito, a 3-dimensional latent-space autoencoder, and a frozen-encoder regression head. Includes reproduction of Pan et al. (2024) &amp; robustness analysis w/ Gaussian noise.
